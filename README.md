# Online Retail Sales Analysis

## Overview
End-to-end data analysis project on 500K+ real retail transactions from a UK-based online store (2010–2011). This project covers data cleaning, exploratory data analysis, SQL-based business queries, RFM customer segmentation, and an interactive Power BI dashboard.

## Tools Used
- Python (pandas, matplotlib, seaborn)
- SQL (SQLite via Python)
- Power BI

## How to Run

### 1. Download the Dataset
- Go to [Kaggle](https://www.kaggle.com)
- Search: `Online Retail UCI`
- Download `online_retail.xlsx` and place it in the `data/` folder

### 2. Install Dependencies
```bash
pip install -r requirements.txt
```

### 3. Run the Analysis
```bash
python analysis.py
```

This will:
- Clean the raw data
- Generate all visualizations (saved to `visuals/`)
- Run SQL queries and print results
- Export cleaned CSVs to `data/`

### 4. Power BI Dashboard
- Open Power BI Desktop
- Import `data/cleaned_retail.csv`, `data/rfm_segments.csv`, `data/monthly_summary.csv`
- Build visuals as described in the dashboard section below

## Key Findings
- 85% of revenue originates from the United Kingdom
- Revenue peaks in November, indicating strong seasonal demand
- Top 10% of customers generate ~50% of total revenue
- Mid-week (Tue–Thu) sees the highest order volumes
- Champions segment customers have 3x higher average order value than Lost customers

## Dashboard Pages
1. **Overview** — KPI cards, monthly revenue trend, top countries, top products
2. **Customer Analysis** — RFM segments, revenue by segment, top customers
3. **Product & Time Analysis** — orders by day, revenue matrix, slicers
